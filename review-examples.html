Review Function Examples

This page demonstrates how to use the review submission functions in your application.

1. Simple Review Submission

Submit a review programmatically with order ID, rating, and description.

// Basic usage await addReview(123, 5, "Great food and excellent service!"); // With error handling try { await addReview(123, 5, "Great food and excellent service!"); } catch (error) { }

2. Review with Modal Interface

Open a modal for users to submit reviews interactively.

// Open review modal for an order addReviewWithModal(123); // With order information addReviewWithModal(123, { orderInfo: { id: 123, order_date: "2024-01-15", total_amount: 25.99 } });

3. Advanced Review with Callbacks

Use callbacks to handle loading states, success, and error scenarios.

await addReview(123, 4, "Good experience overall", { onLoading: (isLoading) => { // Update UI loading state }, onSuccess: (response) => { // Handle success (e.g., refresh data) }, onError: (error, message) => { // Handle error (e.g., show custom message) }, successMessage: "Thank you for your feedback!" });
